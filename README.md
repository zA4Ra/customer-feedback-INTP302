# Customer Feedback Sentiment Dashboard

## Overview

A web application that analyzes customer feedback using Azure AI Language.

## Features

- Submit customer comments
- Sentiment analysis
- Key phrase extraction
- Dashboard analytics

## Tech Stack

- Frontend
- Backend
- Azure App Service
- Azure AI Language
- Azure SQL Database
